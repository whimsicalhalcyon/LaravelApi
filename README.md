# Система обратной связи и управления новостями

 

## Описание

 

Веб-приложение для обработки обращений пользователей и управления новостями через административную панель.

 

Пользователь отправляет сообщение через форму, администратор обрабатывает его, отвечает и управляет статусами. Также реализовано добавление и управление новостями.

 

---

## Функционал пользователя

Пользователь заполняет форму обратной связи:

-  Email 
-  Сообщение 
-  Категория: 
  -  Жалоба 
  -  Предложение 
  -  Вопрос 
  -  Проблема 

 

После отправки:

 

-  Сообщение сохраняется в системе 
-  Пользователь получает ответ на почту (после обработки) 
-  При смене статуса приходит уведомление 

 

---

 

## Функционал администратора


### Работа с сообщениями

-  Просмотр всех сообщений 
-  Ответ пользователю (с отправкой email) 
-  Изменение статуса: 
  -  Новое (по умолчанию) 
  -  На рассмотрении 
  -  Выполнено 
-  Удаление сообщений 


### Статистика

-  Общее количество сообщений 
-  Количество новых сообщений 

### Фильтрация

-  По дате 
-  По статусу 
-  По категории сообщений 

 

---

 

## Аутентификация

### Регистрация

-  ФИО (полностью) 
-  Логин: 
  -  Только латиница 
  -  Минимум 5 символов 
  -  Минимум 2 цифры 
-  Пароль: 
  -  Минимум 8 символов 
  -  Заглавные и строчные буквы 
  -  Минимум 2 цифры 
  -  Спецсимволы (@#$&) 
-  Подтверждение пароля 

 

### Авторизация

-  Почта
-  Пароль 

 

---

 

## Общая статистика

-  Общее количество сообщений 
-  Количество непрочитанных сообщений 
-  Общее количество новостей

## API

### Отправка сообщения

POST /api/messages

```javascript
{
  "title": "Тема",
  "email_send": "test@mail.com",
  "message": "Текст сообщения",
  "category_id": 1
}
```

## Email-уведомления

-  Ответе администратора

php artisan migrate

npm install
